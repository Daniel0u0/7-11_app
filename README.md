7-Eleven App (Unofficial)

An unofficial 7-Eleven Android application developed using Java.

Project Introduction

This is an unofficial 7-Eleven convenience store mobile application designed to provide a convenient 7-Eleven-related service experience. Currently, user login and registration functions are implemented, along with a 2x2 grid layout UI design for the main screen.

Note: This project is a personal final year project only and is not an official 7-Eleven application. No official services or guarantees are provided.

Features

- User Login and Registration

- Main screen uses a 2x2 grid layout

- Pure Java Android App

(The project is still under development, more features will be added later)

Technology Stack

- Language: Java

- Build Tool: Gradle (Kotlin DSL)

- Platform: Android

How to Build and Run

Prerequisites

- Android Studio (latest version recommended)

- JDK 11 or above

Steps

1. Clone this project:

```
git clone https://github.com/Daniel0u0/7-11_app.git

```

2. Open Android Studio, select "Open an existing project" and choose the project folder.

3. Wait for Gradle to complete synchronization.

4. Connect to the Android emulator or physical device, and click Run to execute the application.
